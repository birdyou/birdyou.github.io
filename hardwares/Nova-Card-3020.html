Nova-Card-3020

硬件实物照

硬件基本信息

项目 内容 备注 说明
接口 DIP24 2.54mm标准间距，双列直插 能直接用于万能板，面包板，排针，杜邦线
驱动能力 1A 板载4个MOS管 无需外部电路，直接驱动马达，LED灯珠，蜂鸣器等
ADC 6路，10bit N/A 可外接多达6个模拟器件，如模拟式传感器，键盘等
GPIO 12路IO N/A 所有IO均可自由映射为输入/输出，SPI、I2C、PWM
电源输入 3v到19v 4节五号电池(6v)，锂电池(4v)，常用变压器(5v,6v,9v,12v)，汽车电路(12v) 能直接适配多种电压，方便应用
电源输出 3v 2路3v输出，方便驱动3v数字/模拟外设 N/A

软件基本信息

项目 内容 备注 说明
用户代码区 32k代码，4k内存 模块可二次开发 无需外接MCU即可运行用户自己的程序
任务调度 异步多任务 可调度最多32个异步任务 相比MCU常见的轮询式编程，异步编程效率更高、更省电.
固件烧录 有线/无线双模式 有线模式: JLINK烧录机，速度快; 无线模式: 通过手机无线烧录，更方便 方便灵活的烧录方式.
API 标准C API 良好封装的API，方便调用 无需操作寄存器，不用了解蓝牙底层协议，即可开发蓝牙智能产品.
