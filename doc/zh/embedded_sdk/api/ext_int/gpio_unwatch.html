解除回调函数

函数声明

void gpio_unwatch(uint8_t pin);

函数功能

解除已经注册的 GPIO 外部触发事件。

函数参数

参数 数据类型 说明
返回值 无 无

函数举例

    //当前函数
    void foo()
    {
        ...
        
        //取消对先前引脚的监听
        gpio_unwatch(8);
        
        ...
    }
