信号强度事件

函数声明

public void onRemoteRssi(int status, int rssi);

函数用途

执行获取蓝牙设备信号强度操作时，会触发此回调。

函数参数

参数 数据类型 说明
status int 获取蓝牙设备信号强度操作的状态。SUCCESS 或 ERROR
rssi int 蓝牙设备的信号强度
